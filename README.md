# moonbit-js-ffi/fs


## Description

moonbit-js-ffi/fs is a collection of filesystem for the MoonBit JavaScript FFI library.

## Installation

 moon add moonbit-js-ffi/fs

## Usage

```rust
fn main {
  let a = @fs.readFile("./package.json")
  println(a)
}

```
